ДАТА СНЯТИЯ ПОКАЗАНИЙ
Дата будет добавлена ко всем показаниям при отправке.
КАК СНЯТЬ ПОКАЗАНИЯ
1. Открывается фильтр НЕ ЗАПОЛНЕНО — видны только незаполненные счётчики
2. Нажмите на нужный счётчик — откроется форма ввода
3. Введите показание цифрами
4. Нажмите СОХРАНИТЬ
5. Счётчик станет жёлтым — ожидает отправки
6. Когда есть интернет — нажмите ОТПРАВИТЬ вверху
7. Счётчик станет зелёным — отправлено ✓
НЕТ ДОСТУПА К СЧЁТЧИКУ
Если арендатора нет и счётчик недоступен:
→ Нажмите кнопку НЕТ ДОСТУПА вместо ввода показания
→ Счётчик отметится жёлтым
→ В базу уйдёт пометка без показания
ФИЛЬТРЫ
НЕ ЗАПОЛНЕНО — только счётчики без показания. Открывается по умолчанию.
ЭЛЕКТРО — все электросчётчики (Э1А, Э2...)
ВОДА — все водосчётчики (В1Гл, В2...)
ВСЕ — все счётчики. Видно кто заполнен а кто нет.
ВКЛАДКА ОЧЕРЕДЬ
Здесь видны все введённые показания.
⏳ ожидает — ещё не отправлено
✓ отправлено — успешно отправлено

✓ Удалить отправленные — убирает зелёные записи из списка
✕ Очистить все показания — удаляет ВСЕ записи и сбрасывает все метки на счётчиках. Использовать только после отправки всех данных!
НАСТРОЙКИ
Дата снятия показаний — устанавливается автоматически на сегодня при каждом открытии приложения. Можно изменить вручную если снимаете показания за другой день.

URL скрипта — адрес для отправки данных. Не менять без необходимости.
